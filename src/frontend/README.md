# Daily Coupons - Progressive Web App

A daily coupon giveaway web application built with React, TypeScript, and Internet Computer, featuring Internet Identity authentication, coupon history, and brand listings.

## üöÄ Features

- **Daily Coupons**: Discover exclusive coupon codes released daily
- **Internet Identity**: Secure authentication using Internet Computer's Internet Identity
- **Coupon History**: Track all your coupons and redemption status
- **Brand Listings**: Browse coupons organized by brand
- **Progressive Web App**: Install on your device for a native app-like experience

## üì± Installing as a PWA

### On Android (Chrome)

1. Open the app in Chrome browser using your production URL
2. Tap the menu icon (three dots) in the top-right corner
3. Select **"Install app"** or **"Add to Home screen"**
4. Follow the prompts to add the app to your home screen
5. The app will now launch in standalone mode, just like a native app!

### On Desktop (Chrome/Edge)

1. Open the app in Chrome or Edge browser
2. Look for the install icon (‚ûï or computer icon) in the address bar
3. Click the icon and select **"Install"**
4. The app will open in its own window

### On iOS (Safari)

1. Open the app in Safari browser
2. Tap the Share button (square with arrow pointing up)
3. Scroll down and tap **"Add to Home Screen"**
4. Name the app and tap **"Add"**
5. The app icon will appear on your home screen
